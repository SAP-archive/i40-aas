# i40-aas-registry

## Database Connection

Database properties are available as the following environment variables:

```
POSTGRES_HOST
POSTGRES_PORT
POSTGRES_DB
POSTGRES_USER
POSTGRES_PASSWORD
```

The PGAdmin credentials are

e-mail: `admin`
pw: `admin`

## Registry basic auth

Registry admin user is available as the following environment variables:

```
REGISTRY_ADMIN_USER
REGISTRY_ADMIN_PASSWORD
```
